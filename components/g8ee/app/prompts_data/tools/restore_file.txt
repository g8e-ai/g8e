Restore a file to a previous version using a commit hash from the Operator's Ledger. Overwrites current file. Restoration recorded as a new mutation for audit.

<parameters>
- file_path, commit_hash: REQUIRED
- target_operator: REQUIRED when multiple operators bound
</parameters>

<workflow>
1. fetch_file_history(file_path="...") — show available versions to user
2. restore_file(file_path="...", commit_hash="...") — restore selected version
</workflow>
