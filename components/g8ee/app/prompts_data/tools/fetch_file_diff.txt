Retrieve file diffs from the Operator's Sentinel-scrubbed vault. Always reads from scrubbed vault — never raw.

<parameters>
- diff_id: retrieve full diff content for a specific diff
- web_session_id: list all diffs for a session (at least one required)
- file_path: filter by path (with web_session_id only) | limit: default 50 | target_operator: REQUIRED when multiple operators bound
</parameters>

<returns>
By diff_id: id, timestamp, file_path, operation, diff_stat, diff_content (full git diff), diff_size
By web_session_id: array of diff summaries (no full content), total, web_session_id
</returns>

<workflow>
1. fetch_file_diff(web_session_id="...", file_path="...") — get diff list
2. fetch_file_diff(diff_id="...") — get full content for specific diff
</workflow>

<when_to_use>
- User asks "what changes were made?" or needs to audit file modifications
- Debugging by reviewing exact diff content
</when_to_use>
