Read text file contents. Preferred over `cat` — no shell process.

<parameters>
- path: REQUIRED — absolute or relative to operator working directory
- max_size: default 100KB, max 1MB (truncated if exceeded)
- target_operator: REQUIRED when multiple operators bound (hostname, operator_id, or index)
</parameters>

<behavior>
Binary files rejected. Large files truncated with warning. Symlinks followed.
</behavior>

<sentinel_scrubbing>
File content is Sentinel-scrubbed. See run_commands_with_operator for full placeholder reference.
</sentinel_scrubbing>
