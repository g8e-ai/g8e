Getting Started
1 arrow_downward
Download the Operator
Run the deployment script on your target host to download and launch the operator binary.
curl -fsSL http://<host>/g8e | sh -s -- <token>
2 arrow_downward
Authenticate the Session
Establish a secure session and verify your identity using the CLI.
./g8e login --device-token <token>
3 arrow_back
Manually Bind to Web Session
Explicitly bind the operator to your web session in the dashboard. This manual step is a critical layer of human intent in the g8e security model.
g8ep is ready. Click the link icon to begin.
