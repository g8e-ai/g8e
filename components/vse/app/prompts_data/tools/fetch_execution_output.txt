Retrieve raw command output from the Operator's local LFAA storage. Use ONLY when you need raw unredacted output or must re-examine output from a previous command. The scrubbed output in command results is sufficient for normal analysis.

<parameters>
- execution_id: REQUIRED — unique ID from command result metadata
- target_operator: hostname, operator_id, or index (REQUIRED when multiple operators bound)
</parameters>

<when_to_use>
- User asks about output from a previous command
- Output was truncated and you need the full content
- Comparing output across multiple executions
</when_to_use>

<sentinel_scrubbing>
Output is Sentinel-scrubbed. See run_commands_with_operator for full placeholder reference.
</sentinel_scrubbing>
