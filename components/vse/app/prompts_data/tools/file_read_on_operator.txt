Read content from a file on the g8e Operator system.

<parameters>
- file_path, justification: REQUIRED
- start_line, end_line, max_lines: for large files
- target_operator: REQUIRED when multiple operators bound
</parameters>

<behavior>
Always read a file before attempting to update it.
</behavior>

<sentinel_scrubbing>
File content is Sentinel-scrubbed. See run_commands_with_operator for full placeholder reference.
</sentinel_scrubbing>
