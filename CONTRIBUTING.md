# Contributing to g8e

g8e is an open source project.

We appreciate your help!

## Filing issues

When [filing an issue](https://github.com/g8e-ai/g8e/issues/new), make sure to answer these five questions:

1. What version of g8e are you using (`./g8e version`)?
2. What operating system and processor architecture are you using?
3. What did you do?
4. What did you expect to see?
5. What did you see instead?

General questions should go to danny@g8e.ai.
